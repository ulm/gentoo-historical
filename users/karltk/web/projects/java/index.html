Java work

The Gentoo Java Herd are working on the following, in no particular order:

  • List of bugs
  • Setting up a wiki
  • Finishing our new ebuild policy
  • Figuring out a good system for servlets and webapps, contact axxo.
  • Writing ebuild helper scripts and stuffing them in dev-java/javatoolkit
  • Getting all java packages to run on a FOSS java stack.
  • Recruiting new Java developers for Gentoo, as powerusers giving feedback and as prospective maintainers
