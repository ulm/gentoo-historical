webapp-config - snapshot releases
=================================

Copy contents of:	into:

sbin/			/usr/sbin/
config/			/etc/conf.d/
eclass/			/usr/portage/eclass/
lib/			/usr/lib/webapp-config/

An ebuild for webapp-config is now available in Portage.

Example ebuilds can be found in examples/.  The plan is to provide example
ebuilds for:

* phpmyadmin
* fudforums
* twiki
