Table Listing:
A table listing is a table that lists out objects with a set of properties,
such as users or scripts or pretty much any kind of data record. A table
listing allows the following functionality:
 - Displays a set of data records
 - Paging through the data records
 - User specified:
  - number of rows
  - sort field
! - secondary sort field
  - ascending or descending order
! - hide/unhide columns
  - link to a more detailed page on that record

Common Page:
This is basically just a general page used for pretty much everything in the
UI. It's features include:
 - Display of error message
 - Display of notice message.
